What is GitHub ?

GitHub is an online platform used to collaborate, share and work on developping project via repositories. Depending on the level of access we have on projects, we can directly bring modifications on the code or ask for the administration of the project via a pull request to review these modification before applying them if judged beneficial. We can keep track on the modifications brought by other contributors, propose some new code with comments and explanation, react to the other's code, but also create portfolio and personal projects for future references. In a way, it also works as a professionnal social media. 

We learnt about GitHub and how to use it in our Computer Tools Class with Kaveh Bakhtiyari.

How C# works ?

First we write the code. Then it is checked by the compiler to see if there is any errors in it and informs us if it is the case. The code is then converted to intermediate language (IL) code in order to be interpreted by any machine having .NET runtime installed. Next, the CLR (Common Language Runtime) calls the JIT compiler to convert it to machine code.
