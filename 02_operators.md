Section 1 - Concept of Operators

An operator is a symbol used to perfome operations on values or variables. 
There are four categories of operators. The arithmetic operators (/, * , -, +, %) are used to operate calculations while the comparison operators (>, <, ==, !=, >=, <=) are used to compare variables or values.
We also have logical operators (&&, ||, !) used with boolean expressions to combine conditions, and finally the assignment operator (=) used to assign a value to a variable.


Section 2 - Syntax

Arithmetic + assignment operator example:
int a = 3 + 7; 
--> First "+" adds 7 to 3, giving 10, and then "=" gives the value of 10 to the variable "a" of integer type.
