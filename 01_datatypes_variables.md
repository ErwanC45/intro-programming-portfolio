Section 1 - Concept of Variable

A variable is like a container in memory. We first need to give it a name and a type. It's called declaration. 
A variable can be of many types : integer (byte, short, int, long, can also be unsigned or signed), character, string, boolean, float (float, double, decimal). 
It is important to give the type of the variable, because it conditions the space in memory that the program will reserve for this variable. For example, an int takes 4 bytes while a short takes 2 and a long takes 8.

A variable can be assigned a value (initialization) or left empty for later purposes. This variable can then be used in many ways by the program, for example in calculations.

For instance, we can declare and initialize two integers (a and b) and add them to produce a third one (c). 
But we can change the value of a variable without changing its name, making it dynamic and preventing hardcoding. In my previous example, if we change the value of a or b, it will also change the value of c.

Section 2 - Syntax

Declaration:\n

Initialization:

Declaration and initilization:

Example of calculation:
